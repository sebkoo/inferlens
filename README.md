# Inferlens

On-device image classification that logs every run, to measure Core ML vs TensorFlow Lite
on your own iPhone. Work in progress — see [docs/ROADMAP.md](docs/ROADMAP.md) for the
commit ladder and [docs/adr/](docs/adr/) for the decisions behind it. The real README
(latency table, state-machine diagram, methodology) lands at the end of the ladder.
